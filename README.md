# SelTextCalculator
This plugin can sum or multiply numbers on selected text in Notepad Plus Plus.

## KeyBindings
- Ctrl + Alt + A : Sum all numbers in selected text;
- Ctrl + Alt + M : Multiply all numbers in selected text;
